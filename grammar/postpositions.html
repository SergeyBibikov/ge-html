Послелоги

    Пишутся слитно С именительным или дательным п-м:
  • ვით - словно, как (ადამიანივით - словно человек, მზესავით - похожий на солнце)
    • С дательным п-м:
  • ში - в (ბათუმსი - в Батуми)
  • ზე - на (მაგიდაზე - на столе)
  • თან - у, около (სახლთან - у дома)
    • С творительным п-м:
  • დან - из, от места (თბილისიდან - из Тбилиси)
  • ურთ - с кем-то (შვილებითურთ - с детьми)
    • С обстоятельственным п-м:
  • მდე - до(о времени или месте) (ქალაქამდე - до города)
    • С родительным п-м:
  • ებრ - вроде, наподобие (მთვარესებრ - подобно луне)
  • გან - от человека (მეგობრისაგან - от друга)
  • თანავე - сразу после (მოსვლისთანავე - по прибытии)
  • კენ - в сторону, в направлении (თბილისისკენ - в направлении Тбилиси)
  • თვის - для, ради (ძმისთვის - для брата)
    Пишутся отдельным словом после основного слова
  • С дательным п-м:
    შუა - по середине, между (ქალაქებს შუა - между городами)
  • შორის - среди, между (სოფლებს შორის - между деревнями)
    • С родительным п-м:
  • შესახებ - о (წიგნის შესახებ - о книге)
  • მიხედვით - согласно, в соответствии с (კანონის მიხედვით - по закону)
  • მიუხედავად - несмотря на, вопреки (სირთულეების მიუხედავად - несмотря на трудности)
  • ნაცვლად - вместо (რვეულის ნაცვლად - вместо тетради)
  • მაგიერ - взамен, вместо (წიგნის მაგიერ - вместо книги)
  • გარეშე - вне, без (ფანქრის გარეშე - без карандаша)
  • გამო - ввиду, по причине (მეზობლის გამო - из-за соседа)
  • გარდა - за исключением (ბავშვების გარდა - за исключением детей)
  • ჩათვლით - включая (მოხუცების ჩათვლით - включая стариков)
  • მიერ - выполненный кем-то (პუშკინის მიერ დაწერილი - написанный Пушкиным)
  • მიმართ - направленный на кого-то, к кому-то (ქალებისა და გოგონების მიმართ ძალადობა - жестокость в отношении женщин и девушек)
  • შემდეგ - после, спустя (წუთის შემდეგ - спустя минуту)
  • გულისათვის - из-за, за что-то (типа оплаты) (არაფრის გულისათვის - ни за что)
